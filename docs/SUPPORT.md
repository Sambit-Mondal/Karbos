# Support

## How to get help

If you need help with this project, here are a few ways to get in touch:

*   **GitHub Issues**: For bug reports and feature requests, please open an issue on our [GitHub repository](https://github.com/Sambit-Mondal/Karbos/issues).
*   **Discussions**: For general questions and discussions, please use the [GitHub Discussions](https://github.com/Sambit-Mondal/Karbos/discussions) tab.

## Community

We welcome contributions from the community! Please see our [CONTRIBUTING.md](../CONTRIBUTING.md) file for guidelines on how to contribute.
